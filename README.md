***\*摘要\****

 

“为有牺牲多壮志，敢教日月换新天。”红色旅游路线，之所以鲜红，只因那是烈士鲜血铺就。党的十九届五中全会从战略全局出发对推进文化强国建设作出规划设计。在建设文化强国的历史新征程中，我们必须充分认识红色文化在举旗帜、聚民心、育新人、兴文化、展形象中的重要地位和作用。正值建党一百周年，百年岁月波澜壮阔，百年风华泰然沉稳。习总书记在庆祝中国共产党成立一百周年大会上强调：“历史川流不息，精神代代相传。我们要继续***\*弘扬光荣传统、赓续红色血脉\****，永远把伟大建党精神继承下去、发扬光大！”然而多年来，由于其本身的严肃性特征，不同于茶余饭后的闲谈佳话，红色文化与大众之间存在一定的***\*距离感\****。

本项目组在经过一定的文献资料准备和充分的社会调查下，对市面上宣扬红色文化类APP进行比对分析，找出亮点与不足，最终确定用户需求。项目组决定以浙江省在2020中国红色旅游推广联盟年会暨革命精神传承发展现场会上推出的***\*“红色精神”\****之旅、“***\*红色信仰”\****之旅、***\*“烽火岁月”\****之旅、***\*“山海垦荒”\****之旅四条线路为***\*落脚点\****，宣扬百年间发生在之江大地上的红色文化，利用Android开发“之江韫”APP。透过岁月的罅隙，更多的人得以知道这段辉煌而振奋人心的历史，大力弘扬红色文化，拉近其与大众间的距离，为我国文化建设强基固本、壮骨铸魂。

“之江韫”APP，涵盖浙江***\*“红色精神”\****之旅、“***\*红色信仰”\****之旅、***\*“烽火岁月”\****之旅、***\*“山海垦荒”\****之旅四条红色线路，主要包括首页“浙”、“途·征”、“知·揽”、“薪·传”、“思·筑”、“个人主页”六大模块，为用户构建全方位的思维脉络，深入学习浙江红色文化。***\*首页“浙”\****为用户***\*推荐\****最新、最热，及感兴趣的文章，上方的精品画廊，采用图文并茂的方式介绍四条旅游路线，用户通过点击就能够***\*跳转\****到“途·征”模块中对应的路线。***\*“\*******\*每日打卡\*******\*”\****和***\*搜索\****功能，帮助用户养成良好的习惯，快速获取想要了解的内容。***\*“途·征”\****是模块为此APP的核心功能模块，其中详细介绍了四条旅游路线***\*（“红色精神”\****、“***\*红色信仰”\****、***\*“烽火岁月”\****、***\*“山海垦荒”）\****，用户可以在右上方自由切换四条线路对其进行了解，不仅能够通过地图描绘线路的轮廓，还能快速地了解沿途景观，如果感兴趣，还能够进一步深入探索——VR观景带领用户身临其境，优美动听的解说将景区文化娓娓道来。此外，根据用户兴趣，文章设置了***\*“收藏”、“笔记”、“评论”、“分享”\****的互动学习功能——用户可以将阅读的感兴趣文章***\*“收藏”\****，方便下次查看；也可以将学习心得以***\*“笔记”\****的形式记录下来，巩固重难点；如果有感而发，可进行***\*“评论”\****，还能够***\*“分享”\****给其他人，一起体验这份学习的快感。***\*“知·揽”\****涵盖了丰富的学习资源，如影像资料、图文资料、音频资料等。用户也可以通过红色事件、红色人物、历史文献、文艺作品进行分门别类地学习。***\*“薪·传”\****实现***\*论坛功能\****，可发表***\*学习心得\****，可进行***\*问题探讨\****，从而达到巩固所学、提升自我的新境界。***\*“思·筑”\****中有各类精彩小游戏，以此可***\*检验学习成果\****。***\*红色答题、红色拼图、红色朗诵、红色绘画、听红歌，\****丰富多样。更有积分排行榜和荣誉勋章墙，紧张刺激。***\*“个人主页”\****方便用户***\*管理个人信息\****，***\*“收藏”、“笔记”、“评论”、“分享”\****都会体现其中。更提供***\*“我的圈子”\****，让用户掌握发帖状况。

# ***\*第1章 绪论\****

## ***\*1\*******\*.1\**** ***\*选题背景\****

这里的故事，有一个世纪那么长。自1921年中国共产党诞生，实现开天辟地的壮举以来，到如今，巍巍华夏民族栉风沐雨，又走过了一百个年头。“千淘万漉虽辛苦，吹尽狂沙始到金。”岁月洗去了浮世的喧嚣与尘埃，璀璨的沧海遗珠呈现于世人面前，愈加珍贵明丽。

恰同学少年，风华正茂，书生意气。在中国共产党的带领下，衙前农民协会成立，首个《共产党宣言》中译本问世，浙江省一大召开，亭旁起义建立浙江省第一个苏维埃政权。“红色精神”、“红色信仰”、“烽火岁月”、“山海垦荒”，这些携带时代烙印的字眼款款而来，留下了深深的痕迹。

全国百条红色旅游线路，光是浙江就有四条。通过这些线路，我们得以重温红色历史，去了解那一段我们未曾参与却闪光的历史。那些斑驳树影，好像在昭示这波澜壮阔。

为了让更多的人去了解这一段跨越百年的历史，也为了让这段历史更加生动形象地展现在人们面前，***\*我们以浙江四条红色旅游路线为基础，进行了时间和空间的双重还原，搜集了百年来发生在之江大地上那鲜妍明亮的中国红，红色影像、红色事件、红色人物、重要文献、文艺作品，共同构建了这一抹独具特色的“之江韫”。\****

## ***\*1\*******\*.2\**** ***\*目的和意义\****

从全国百条红色旅游线路出发，最终落脚于之江大地，以“红色精神”、“红色信仰”、“烽火岁月”、“山海垦荒”为关键词，揭开浙江百年来波澜壮阔的红色故事。

百年间山河激荡、风起云涌，漫漫峥嵘岁月像是一部纪实的影片，将那段波澜壮阔的历史一帧帧放映。“之江韫”给人以直观的感知体验和真实的事件还原，通过此，更多的人能够去了解在中国共产党的领导下，浙江一步一步走向美好的风雨历程。这里不仅可以纵览四条旅游路线的全貌，还能了解到每个标志性地点的详细信息，图文并茂。也可以与伟人们，英雄们进行精神交流，通过精彩纷呈的文艺作品和珍贵的文献史料，找到情感的共鸣。

在论坛上，大家可以各抒己见，针对相关事件发表不同的观点。在激烈的讨论中，将知识掌握得更牢固，消化得更彻底。在小游戏中学与练，在收藏中温故而知新，渐入佳境。



# ***\*第2章 开发工具与系统架构\****

## ***\*2\*******\*.1\**** ***\*开发模式\****

本项目采用MVVM架构。

MVVM框架的优势：
	1. View（ViewController）通过对ViewModel中的数据进行绑定来更新界面，不用通过逻辑或者条件判断来更新view，大大降低了复杂交互时出bug的几率。
	2. View（ViewController）中代码简洁，后期的维护和优化比较容易。

3.MVVM框架是在MVC的基础上演化而来，MVVM想要解决的问题是尽可能地减少Controller的任务。

<img width="438" alt="image" src="https://user-images.githubusercontent.com/82577086/230064826-20e2328c-ad4b-4e44-ade5-5d8f760c7d44.png">

***\*Model：\****程序中要操纵的实际对象的抽象
View(ViewController)：MVVM中的View不再是UIView的子类，而变成了UIViewController的子类。这里的View实际上就是MVC中剥离了处理呈现View逻辑部分的Controller，因此它仍然有各种UIView的属性，仍然有ViewController的声明周期的各种方法，但是这里的Controller不再负责数据的请求以及处理逻辑，因此不再臃肿。
	***\*ViewModel：\****MVVM中，ViewModel代替了MVC中的Controller成为了协调者的角色，ViewModel被View（ViewController）持有，同时持有者Model。数据请求以及处理逻辑都放在ViewModel中，View(ViewController)就瘦了下来。

## ***\*4\*******\*.4\**** ***\*实现程度\****

项目组通过分析用户需求，设计“之江韫”APP的系统架构，再利用visio和ioDraw设计简单的逻辑结构；在逻辑结构较为完善的条件下，进行后台数据库搭建与管理。目前APP的后台已经基本实现，“之江韫”APP已经完成了90%的功能。

剩余少数未完成的功能主要是：根据用户的学习情况和兴趣爱好，推荐用户喜欢的内容，增加用户的体验。接下来团队将致力于剩下功能的实现，并会不断研究Android开发新功能，深入了解不同用户的需求，努力提升我们的产品质量，致力于打造浙江红色教育APP精品，宣传“红色精神”，让参与使用者能够深刻感受到“信仰”的力量，哪怕在那一段硝烟弥漫、“烽火”连天的“岁月”里，依然不放弃艰苦卓绝的斗争，追逐光芒前行。英勇前辈们筚路蓝缕的“垦荒”步履，终换来这韫色之江“山”河无恙，“海”纳百川。



# ***\*第\*******\*5章 系统创新\****

## ***\*5\*******\*.1\**** ***\*功能创新\****

“之江韫”进行了人性化设计，尊重新用户对新鲜事物的猎奇心理和试探心理，给与他们足够的空间。在纯浏览的情况下，用户不需要进行注册和登录。也就是说，用户不需要登录，就能使用应用中的几乎全部的学习资源，也能够在学习之余，在外面精心设计的配套小游戏里放松身心，巩固新知识。同时我们也竭尽所能给用户呈现出完整且丰富多样的学习资源，让学习更全面，轻松掌握全局。

我们希望使用该应用的每一个用户，都能够在其中找到自己想要的资料，能激发他们的兴趣，在快乐中学习新知识。“随风潜入夜，润物细无声。” 希望“之江韫”能够发挥其作教育类App的作用，给使用者带去如沐春风的体验感。让他们对于知识的学习不再停留于简简单单的被动接受，而是参与其中，更多样，更主动。当然，该应用有足够的用户黏性，资源推送、每日打卡、文章收藏、阅读标记、笔记整理和分享，以及点赞、讨论，是我们为每一位用户的个性化定制。在学中练，在玩中学，错题整理复习巩固，积分排行紧张刺激。

## ***\*5\*******\*.2\**** ***\*应用创新\****

景与学相结合，学与练相辅相成，让用户能在体验VR红色景点中感受到红色文化，同时在学习巩固练习的时候能够有成就体验，完成对应的任务用户可获得对应的奖章和积分，同时可查看已获得的积分和排名。



# ***\*第\*******\*6章 应用前景\****

## ***\*6\*******\*.1\**** ***\*市场前景\****

“之江韫”，浙江红色教育类App，以浙江省四条红色旅游路线为基础，依次罗列沿途景点绘制地图，给人一种空间上的形象感和层次感，影像资料完备，红色事件、人物、文献、作品齐全，分门别类可供用户使用，也可在论坛上发表见解、互动讨论，通过趣味小游戏还能够巩固知识。***\*该应用功能完备，兼具学、练、测、评，极具使用价值和良好的体验感，市场前景广泛。\****

***\*第一，具有良好的社会适应性。\****在如今疫情大环境下，线上教学的必要性与优势日趋凸显。同样，一款足不出户就能使用的教育类移动应用也有着举足轻重的作用。使用者只要通过手机和互联网就能够进行学习，十分便捷。

***\*第二，紧随当下时代的脚步。\****当前，我们已经进入了数据时代，对大数据的分析和其在生活中的应用已屡见不鲜。通过对用户的学习时间段统计、着重或经常性使用的学习资源等等，我们可以为用户制定更加合适的学习计划，推送更多用户感兴趣的文章，从而让他们产生对学习的热情。

***\*第三，学习资源丰富，教育功能突出。\****“之江韫”是一款资源丰富且融会贯通给的综合性App。它涵盖了各种类型的学习资源，丰富多样，其中不仅有最最基础的图片和文本资源，还有视频资源、游戏资源以及VR资源等等，最大程度地满足不同使用者的独特需求。

 

## ***\*6\*******\*.2\**** ***\*推广价值\****

***\*其一，\****“之江韫”作为浙江红色教育类App，以“红色精神”、“红色信仰”、“烽火岁月”、“山海垦荒”为关键词。其在发挥教育学习作用的同时，也能够进行爱国教育与城市文化宣传。践行社会主义核心价值观，将钟林毓秀、人杰地灵的之江大地呈现在更多人面前。

***\*其二，\****“之江韫”内容丰富、创意独特，有其不可替代性。在将来还可以开发配套的小程序和专属网站，资源互通、优势互补，打造“一站式”学习。

***\*其三，\****“之江韫”以浙江省四条红色旅游路线为基础，可以与沿途景点合作，进一步发展旅游业，打造红色旅游。使用者可以通过该应用进行景点地位和导航，也可以考虑某些景点的门票预定等，提供各种精彩周边，旅行者还可以在上面分享旅游攻略，了解该景点的相关知识，体验不一样的红色旅行。

***\*另外，\****我们还能够根据不同的节气和假日，开展线上线下的活动。如，在风清气朗的春三月一起携手踏青，用脚步丈量“信仰”的宽度；在举国欢庆的国庆佳节，为祖国献上精美的画作；荡气回肠的书法，紧张刺激的知识竞赛……用户可以根据自己的兴趣报名。在精彩的比拼中发扬红色精神，传播社会正能量。
